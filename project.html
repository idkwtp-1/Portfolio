Interactive Game for Kids

This project involves an interactive game designed for children. The game features multiple mini-games that randomize each time, ensuring a fresh experience. Players earn points based on their performance, and a final score is calculated at the end. The back-end is built using PHP, while the front-end utilizes HTML and CSS for a colorful and responsive interface. A database is integrated to store player progress and scores.

Library Management System

A library management system has been developed to assist librarians in organizing books, newspapers, and magazines. The system allows users to add new items, search for them by ID, update details, or delete entries as needed. Java is used for both the back-end and front-end (with Java Swing), while DBeaver manages the database. The interface is designed to be simple and user-friendly, making it accessible for all users.

Sharing Recipe

This project is a **recipe-sharing platform** that allows users to **browse, share, and manage recipes**. Anyone can explore recipes without an account, but **registered users can create an account, log in, share their own recipes, and connect with others**. The front-end is built with **HTML, CSS, and JavaScript** for a smooth user experience, while the back-end, powered by **Node.js**, handles authentication, recipe management, and interactions. An **SQL database** securely stores user accounts, and recipes, ensuring seamless access and organization.
